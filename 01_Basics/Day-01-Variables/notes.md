# 📘 Day 1: Variables & Input/Output

## 🧠 What is a Variable?

A variable is like a container that stores data.

Example:
name = "Usha"
age = 20

Here:
- name stores "Usha"
- age stores 20

## 📊 Data Types

- String → "hello"
- Integer → 10
- Float → 3.14
- Boolean → True / False

## 🖨️ Output

print("Hello World")

## ⌨️ Input

name = input("Enter your name: ")
print("Hello", name)

## 🔁 Type Casting

age = int(input("Enter your age: "))
